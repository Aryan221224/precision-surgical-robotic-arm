# Precision 2-DOF Surgical Robotic Arm  
### Laparoscopic Tool Positioning System  

A complete **mechatronics + control systems project** simulating a surgical robotic subsystem for precise tool positioning.

---

## 🔬 Overview
This project implements a **2-DOF planar robotic arm** designed for **minimally invasive surgery**, focusing on:

- High precision (±0.5 mm accuracy)
- Smooth low-vibration motion
- Safety-critical torque limiting
- Real-time control at 1 kHz

---

## ⚙️ Key Features

### 🧩 Full Mechatronics Stack
- Mechanical design (link lengths, workspace)
- Torque analysis with safety factor
- Motor selection (BLDC + planetary gearbox)
- Control system architecture

### 🧠 Advanced Control System
- PID + Gravity Compensation Feedforward
- 1 kHz real-time loop
- Anti-windup + torque limiting
- Singularity detection
